Clique em "Baixar arquivos padrão" para baixar os modelos de arquivos corretos que devem ser utilizados na carga dos comprovantes. Você não precisa utilizá-los, mas serão úteis para conferir se os seus arquivos estão de acordo com o esperado pelo sistema;
2º
Antes de importar o arquivo, o sistema irá realizar algumas verificações para garantir que a carga dos comprovantes ocorra com sucesso:
* Extensão dos arquivos: O sistema só aceitará a importação de arquivos pdf e xlsx;
* Tamanho dos arquivos: Será permitida a importação de arquivos com até {{tamanhoMaximoArquivosAnexos}}MB (soma do tamanho dos dois arquivos importados);
* Arquivo vazio: Se o arquivo xlsx estiver vazio, o sistema irá alertá-lo para que verifique antes de realizar o agendamento da carga dos comprovantes.
* Layout da base SAP: Será verificado se a base SAP importada possui a quantidade de colunas esperadas pelo sistema, conforme disponibilizado nos arquivos padrão.
