1º
Os arquivos físicos que serão anexados aos processos deverão ser copiados para uma pasta específica no NAS configurada no parâmetro jurídico: SERV_CARGA_DOCUMENTOS.
2º
Clique em "Baixar arquivos padrão" para baixar os modelos de arquivos corretos que devem ser utilizados na carga dos comprovantes. Você não precisa utilizá-los, mas serão úteis para conferir se os seus arquivos estão de acordo com o esperado pelo sistema.
3º
Antes de importar o arquivo, o sistema irá realizar algumas verificações para garantir que a carga dos comprovantes ocorra com sucesso:
* Extensão dos arquivos: O sistema só aceitará a importação de arquivos CSV;
* Tamanho dos arquivos: Será permitida a importação de arquivos com até {{tamanhoMaximoArquivoAnexo}}MB;
* Arquivo vazio: Se o arquivo CSV estiver vazio, o sistema irá alertá-lo para que verifique antes de realizar o agendamento da carga dos comprovantes.
* Layout do arquivo: O sistema irá validar se a planilha importada possui mesma quantidade que a planilha padrão.
4º
A planilha deverá conter 3 colunas, sendo:
* Código: Informar o código interno do processo;
* Código tipo de documento: Deverá ser inserido o código do tipo de documento;
* Caminho/Nome do documento: Na coluna em questão deverá ser informado o nome do documento com o seu caminho completo.
Ex.: Users\Exemplo\Documentos\documento.csv
