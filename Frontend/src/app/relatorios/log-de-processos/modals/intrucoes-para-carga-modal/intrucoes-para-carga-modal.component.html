Siga as instruções abaixo para carga dos códigos internos:
1º
Clique em "Baixar arquivo padrão" para baixar o modelo de arquivo correto que deve ser utilizado na carga dos códigos internos dos processos. Você não precisa utilizá-los, mas serão úteis para conferir se os seus arquivos estão de acordo com o esperado pelo sistema;
2º
Antes de importar o arquivo, o sistema irá realizar algumas verificações para garantir que a carga dos códigos internos dos processos ocorra com sucesso:
* Extensão do arquivo: O sistema só aceitará a importação de arquivo csv;
* Tamanho do arquivo: Será permitida a importação de arquivos com até 10MB;
* Arquivo vazio: Se o arquivo csv estiver vazio, o sistema irá alertá-lo para que verifique antes de realizar o agendamento da carga.
