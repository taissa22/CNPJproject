Siga as instruções abaixo para a carga de documentos
1º
Clique em "Baixar arquivos padrão" para baixar o modelo de arquivo correto que deve ser utilizado para migração dos pedidos. Você não precisa utilizá-lo, mas será útil para conferir se o seu arquivo está de acordo com o esperado pelo sistema;
2º
Antes de importar o arquivo, o sistema irá realizar algumas verificações para garantir que a migração de pedidos ocorra com sucesso:
* Extensão do arquivo: O sistema só aceitará a importação de arquivos CSV;
* Tamanho do arquivo: Será permitida a importação de arquivos com até {{tamanhoMaximoArquivoAnexo}}MB;
* Arquivo vazio: Se o arquivo CSV estiver vazio, o sistema irá alertá-lo para que verifique antes de realizar o agendamento da migração de pedidos.
* Layout do arquivo: O sistema irá validar se a planilha importada possui mesma quantidade de colunas que a planilha padrão.
3º
A planilha deverá conter 2 colunas, sendo:
* Número do Pedido Anterior: Deverá conter o número do pedido que será alterado (campo obrigatório, numérico com até 10 dígitos);
* Número do Pedido Novo: Deverá conter o número do pedido para o qual o pedido anterior será migrado (campo obrigatório, numérico com até 10 dígitos).
